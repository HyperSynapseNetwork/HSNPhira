# HSNPhira

## 如何使用本项目

本项目已经将构建与运行脚本化（大概）。

首先你需要 clone 本项目，并进入项目目录。

### 环境变量

本项目通过环境变量来管理配置项。所有可用环境变量都写在 env.sh 中，你可以自行修改其中的配置，对于其中涉及安全性的重要配置请务必修改。

在进行构建/运行之前，需要先运行下面的命令来加载环境变量：

```bash
source env.sh
```

### 构建

运行脚本 build.sh 即可开始构建。会自动安装依赖（hopefully）。

### 运行

运行脚本 start.sh 即可开始运行。在终端中按 Ctrl+C 可以停止运行。
